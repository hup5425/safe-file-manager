=== 안전 파일 관리자 (Safe File Manager) ===

취약한 WP File Manager(elFinder) 대체용 안전 파일 관리자. 워드프레스 관리자 화면에서 서버 파일을 열람·편집·업로드·다운로드합니다.

== 설명 ==

호스팅사가 취약점(예: CVE-2020-25213 계열, elFinder 커넥터 무인증 원격코드실행)을 이유로 WP File Manager 플러그인을 삭제하는 상황에서, 안전하게 같은 편의를 제공하기 위한 대체 플러그인입니다.

= 안전 설계 =
* 취약했던 elFinder 라이브러리를 전혀 쓰지 않습니다.
* 모든 파일 작업 진입점에서 **관리자 권한(manage_options) + nonce** 를 매번 확인합니다. 무인증 접근 경로가 존재하지 않습니다.
* 모든 경로를 realpath 로 정규화한 뒤 **루트(기본: 워드프레스 설치 폴더) 밖으로 나가지 못하게** 검사합니다(디렉터리 탈출·심볼릭 링크 우회 차단).
* 파일 다운로드도 권한 + nonce 확인 후에만 스트리밍합니다.

= 기능 =
* 폴더 탐색(빵부스러기 내비게이션), 파일/폴더 크기·수정일·권한 표시
* 텍스트/코드 파일 편집(2MB 이하), 저장
* 파일 업로드(다중), 다운로드
* 새 폴더·새 파일 생성, 이름 변경, 삭제(폴더 재귀)
* GitHub 릴리스 기반 자동 업데이트(여러 사이트 일괄 반영)

= 설정 상수(wp-config.php) =
* `SFM_CAP` — 파일 작업 허용 권한(기본 `manage_options`).
* `SFM_UPDATE_REPO` — 자동 업데이트 GitHub 'owner/repo'.
* `SFM_UPDATE_TOKEN` — 비공개 저장소용 read 토큰(공개면 불필요).
* 필터 `sfm_base_dir` — 접근 루트를 특정 폴더로 좁힐 수 있습니다.

== 변경 이력 ==

= 1.0.9 =
* 실수 삭제 방지 — 모든 우클릭 메뉴(오른쪽 목록·왼쪽 트리)에서 "삭제" 항목 제거. 삭제는 맨 오른쪽 작업 열의 삭제 버튼으로만.

= 1.0.8 =
* 업데이트 UI를 버튼 2개로 — 평소엔 "지금 업데이트 확인"만, 새 버전이 감지되면 그 오른쪽에 "설치" 버튼이 생성.
* 오른쪽 목록 다중 선택 — Shift+클릭(범위), Ctrl/Cmd+클릭(개별 토글).
* 우클릭 메뉴에 "속성" 추가(단일·다중 모두) — 선택 항목의 폴더 수·파일 수·총 용량 표시.
* 왼쪽 디렉터리 트리에도 우클릭 메뉴 추가(열기·다운로드·이름변경·삭제·속성).
* 하위 폴더가 없는 폴더는 트리에서 펼침 화살표를 표시하지 않음.

= 1.0.7 =
* 뒤로/앞으로를 브라우저 History(popstate) 기반으로 연동 — 마우스 뒤로가기 버튼·백스페이스로도 폴더 뒤로가기(플러그인 화면을 벗어나지 않음).
* ◀ ▶ 버튼을 경로 표시 바로 오른쪽(왼쪽 영역)으로 이동.
* 폴더 이동 시 오른쪽 목록 깜빡임 제거 — 로딩 표시를 지연 적용하고 같은 폴더 재요청은 무시.
* 선택된 행 색이 마우스 올림 색보다 항상 우선하도록(선택=진한 파랑으로 통일).

= 1.0.6 =
* 오른쪽 목록: 한 번 클릭=선택, 더블클릭=열기(폴더 진입/파일 편집), 여백 클릭도 동작.
* 우클릭 컨텍스트 메뉴 추가(열기·다운로드·이름 변경·삭제).
* 앞으로/뒤로 가기 버튼(◀ ▶) 추가 — 파일 관리자 안에서 방문 기록 이동.
* 왼쪽 트리: 폴더 더블클릭으로 펼침/접힘 토글.
* 작업(맨 오른쪽) 버튼을 우측 정렬해 폴더/파일 행의 삭제 버튼 위치를 정렬.

= 1.0.5 =
* 왼쪽 디렉터리 트리 기본 가로 폭을 500px로 확대(여전히 드래그로 조절 가능).

= 1.0.4 =
* 왼쪽 트리에서 폴더를 누를 때 오른쪽 목록이 깜빡이던 문제 개선 — 기존 내용을 지우지 않고 로딩 중엔 살짝 흐리게만 표시했다가 새 내용으로 한 번에 교체.

= 1.0.3 =
* 업데이트 버튼을 1개로 통합 — "지금 업데이트 확인"을 누르고 새 버전이 있으면 같은 버튼이 "업데이트 설치"로 바뀝니다.
* 자동 업데이트를 항상 켜짐으로 변경(체크박스 제거). 워드프레스가 약 12시간마다 새 버전을 확인해 자동 설치하며, 빠르게 적용하려면 위 버튼을 사용합니다.

= 1.0.2 =
* 설정에 "지금 업데이트 확인" 버튼 추가 — 캐시를 강제로 비우고 즉시 재검사, 새 버전이 있으면 그 자리에서 "설치" 버튼으로 업데이트(플러그인 화면 이동 불필요).

= 1.0.1 =
* 레이아웃을 파일질라식 2단(좌: 디렉터리 트리 / 우: 폴더 내용)으로 변경.
* 각 폴더의 재귀 용량 표시(대용량 트리 보호용 예산 상한, 초과 시 근사치 '+').
* 폴더 단위 다운로드(zip) 지원 — 목록의 폴더 행 및 툴바 "폴더 다운로드" 버튼.
* 편집 팝업이 페이지 로드 시 빈 창으로 항상 떠 있던 버그 수정(CSS [hidden] 우선순위).

= 1.0.0 =
* 최초 릴리스: 파일 탐색·편집·업로드·다운로드·생성·이름변경·삭제, 자동 업데이트.
